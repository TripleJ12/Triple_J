1. ID와 PW를 입력하고 '로그인' 버튼을 누른다. 
	1-1.ID가 없다면 회원가입을 하기 위해 '회원가입' 버튼을 누른다.
	1-2. ID, PW, 이름, 학과를 입력란에 입력하고 '확인' 버튼을 누른다.
2. 검색하고자 하는 도서의 제목을 오른쪽의 '검색'란에 입력한다.


-관리자-
1.도서를 추가하고자할 때, 도서 제목, 출판사, 저자, 고유번호, 대출가능여부, 대출현황을 기입하는 란에 도서의 정보를 기입하고 '추가' 버튼을 누르면, 추가를 실행할 것인지 확인하는 확인 창이 뜬다. 만약 사서가 확인 창의 ‘예’ 버튼을 클릭하면, 오른쪽의 '도서정보'칸에 도서 정보가 추가된다. 만약 사서가 ‘아니오’ 버튼을 클릭하면, 도서 정보의 추가가 취소된다.

2.도서를 수정하고자할 때, 검색 후 오른쪽의 '도서목록'칸에서 도서의 제목을 클릭하면, 그 도서의 도서 제목, 출판사, 저자, 고유번호, 대출가능여부, 대출현황의 정보를 입력하는 빈칸이 채워지고, 수정하고자 하는 부분을 수정한 뒤 '수정' 버튼을 누르면 수정을 실행할 것인지 확인하는 확인 창이 뜬다. 만약 사서가 ‘예’ 버튼을 클릭하면, 오른쪽의 '도서정보'칸에 도서 정보가 수정된다. 만약 사서가 ‘아니오’ 버튼을 클릭하면, 도서 정보의 수정이 취소된다.

3.도서를 삭제하고자할 때, 검색 후 오른쪽의 '도서목록'칸에서 도서의 제목을 클릭하면, 그 도서의 도서 제목, 출판사, 저자, 고유번호, 대출가능여부, 대출현황의 정보를 입력하는 빈칸이 채워지고, '삭제' 버튼을 누르면 삭제를 실행할 것인지 확인하는 확인 창이 뜬다. 만약 사서가 ‘예’ 버튼을 클릭하면, 도서 정보가 삭제된다. 만약 사서가 ‘아니오’ 버튼을 클릭하면, 도서 정보의 삭제가 취소된다.



-학생-
1. 찾고자하는 도서의 제목을 검색란에 입력 후, '검색' 버튼을 누른다.

2. 도서 목록에서 찾고자했던 도서의 제목을 클릭하면 아래 '도서정보'칸에 도서의 정보가 출력된다.

3. 검색한 도서를 대출하고자 할 경우, 해당 도서를 선택하고 그 도서의 대출가능여부가 ‘가능’이라면 아래의‘대출’ 버튼을 클릭한다. 그러면 대출을 할 것인지 확인하는 확인 창이 뜬다. 만약 학생이 ‘예’ 버튼을 클릭하면 대출이 완료된다. 만약 학생이 ‘아니오’ 버튼을 클릭하면 대출이 취소된다. 도서의 대출가능 여부가 ‘불가능’일 경우, 대출이 불가능하다는 오류 창이 뜨며, 대출 기능을 사용할 수 없다.

4. 로그인한 학생이 현재 자신이 대출 중인 도서 목록을 확인하고자 할 경우, ‘나의 대출 현황’ 버튼을 클릭하면 현재 학생이 대출한 도서의 목록이 화면에 나타난다.

5. 학생은 자신이 수행하고자 하는 일을 모두 완료한 후, ‘로그아웃’ 버튼을 클릭함으로써 로그아웃을 한다.
